Perform validation based on list of configs

Parameters

  • configs (list[dict]) – parsed and preprocessed configurations

  • n_jobs – The number of coroutines the function used. The number is limited by the user permissions on the database side. If n_jobs > 1, the function will use coroutine to execute. If n_job = 1, it will use just one thread, default is 1.
